# Universal Web Components

This is a collection of framework-agnostic web components that can be used in any modern JavaScript framework or vanilla JavaScript application.

## Features

- **Framework Agnostic**: Works with React, Vue, Angular, Svelte, and vanilla JavaScript
- **Customizable**: Each component accepts attributes for customization
- **Accessible**: Built with accessibility in mind
- **Responsive**: Designed to work on all screen sizes
- **Event-based API**: Uses custom events for communication
